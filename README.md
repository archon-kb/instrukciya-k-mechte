# Инструкция к мечте — сайт клуба путешествий

Сайт турклуба «Инструкция к мечте»: маршруты, которые ведут люди, а не витрины туров. Клуб — агрегатор личных брендов инструкторов-проводников.

- **Боевой сайт (в разработке):** Vite + React 18 + TypeScript, корень репозитория → деплой на `/preview/`
- **Макет-витрина:** `site-mockup/` (10 статических страниц) → корень GitHub Pages, пока новый сайт не станет эталонным

## Запуск
```bash
npm ci
npm run dev        # dev-сервер (Vite, порт 5173)
npm run build      # прод-сборка в dist/ (base: /instrukciya-k-mechte/preview/)
```

## Структура
```
├── src/                 ← React-приложение (сейчас: hero-превью по референсу)
├── public/              ← боевые веб-ассеты: видео-фон (mp4/webm/poster), фон 1920.webp, hero-SVG
├── site-mockup/         ← статический макет, деплоится в корень Pages
├── tools/               ← CLI-генераторы медиа (Python; pip install -r tools/requirements.txt)
├── media-source/        ← [вне git] невоспроизводимые медиа-мастера; карта — MANIFEST.md (в git)
├── lab/                 ← [вне git] эксперименты и отклонённые ветки генераций
├── знания/              ← [вне git] Obsidian-vault команды: бренд, бизнес, дизайн (вход: знания/_обзор.md)
└── .github/workflows/   ← деплой Pages: макет в корень + приложение в /preview
```

Деплой: push в `main` → GitHub Actions собирает Vite-приложение и публикует вместе с макетом. URL: https://archon-kb.github.io/instrukciya-k-mechte/ (макет) · …/preview/ (приложение).

Внутренние материалы (факты проекта, квест-лог, навигаторы сессий) намеренно не публикуются: вход для команды — `знания/_обзор.md` локально.
